## 1. 翻转链表

- ```java
  class Solution {
      public ListNode reverseList(ListNode head) {
          // pre:
          ListNode pre = null;
          ListNode cur = head;
          ListNode nxt = head;
  
          while(cur != null){
              nxt = cur.next;
              cur.next = pre;
              pre = cur;
              cur = nxt;
          }
          return pre;
      }
  }
  ```
