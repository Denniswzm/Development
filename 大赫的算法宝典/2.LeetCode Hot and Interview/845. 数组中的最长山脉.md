## 1. 双指针

- ```java
  class Solution {
      public int longestMountain(int[] arr) {
          // edge condition
          if(arr == null || arr.length == 0){
              return -1;
          }
  
          // prerequisite
          int len = arr.length;
          int left = 0;
  
          int max = 0;
  
          // two pointers
          while (left + 2 < len) {
              int right = left + 1;
              // 上升
              if(arr[left] < arr[left + 1]){
                  // 移动右侧指针
                  while(right + 1 < len && arr[right] < arr[right + 1]){
                      right++;
                  }
                  // 下降
                  if (right + 1 < len && arr[right] > arr[right + 1]) {
                      // 移动右侧指针
                      while (right + 1 < len && arr[right] > arr[right + 1]) {
                          right++;
                      }
                      max = Math.max(max, right - left + 1);
                  }
              }
              left = right;
          }
          return max;
      }
  }
  ```
